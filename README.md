# NASM Tutorial

A simple tutorial project to learn **NASM (Netwide Assembler)** and x64 assembly programming.

## Installation

1. **Linux (Debian/Ubuntu)**

```bash
sudo apt update
sudo apt install build-essential nasm
```

2. **Clone this repository using ssh or https**

```bash
git clone git@github.com:kamil-klawi/nasm-tutorial.git
cd nasm-tutorial
```

3. **To build and run a program, use the Makefile**

Example "Hello, world!" program:
```bash
make hello
build/hello
```
